## Getting Started

First it is necessary to install NodeJs, Git, npm, yarn.
It would also be nice to have already installed Expo and Insomnia to test the project.


### Installation

1. Clone the repo
```sh
git clone https://github.com/marcosmarcon/covid19-data
```
2. Install packages
```sh
npm install
yarn install
```
